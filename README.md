## Installation :

* `apt update`
* `apt install git -y`
* `git clone git://github.com/Kamran1819G/nexphisher.git`
* `cd nexphisher`
#### > SETUP : `bash setup`
#### > SETUP [TERMUX] : `bash tmux_setup`
#### > Run : `bash nexphisher`

## Single Command :
```
apt update ; apt install git -y ; git clone git://github.com/Kamran1819G/nexphisher.git ; cd nexphisher ; bash setup ; bash nexphisher
```
